The Handbook is the calm reference space for platform structure, governance context and role-aware orientation. Practical resident help belongs in the separate Help Center, so this page stays easy to read and easy to trust.

For residents

Residents should use this page mainly for orientation. When the goal is to complete a task, they should go straight to Help Center from the main menu.

For governance

Admin and Owner can use the Handbook as a clean public-facing reference while keeping editing tools inside Admin where they belong.

The Handbook stays separate from the Help Center on purpose. This keeps the resident experience simpler and keeps governance content more structured.
