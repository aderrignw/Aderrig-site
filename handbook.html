Handbook

The Handbook stays separate from the Help Center. Use this area for organised platform guidance, governance context and role-aware orientation. Use the Help Center from the main menu when you want step-by-step practical help.

Open Help Center Open Dashboard Open Admin

What belongs in the Handbook

Keep this page clear and light. It should guide people without turning into a dense control panel.

OrientationExplain how the platform is structured and where key areas live.
GovernanceKeep role expectations, oversight notes and operating context easy to find.
ReferencePoint people to the right area without opening every control at once.
The Help Center is now its own main-menu page, so practical resident help no longer has to live inside the Handbook.

How to use it

Resident

Use the Handbook for orientation and the Help Center for direct task help.

Admin

Use Admin > Handbook to manage structure and Help Center content without cluttering this page.

Owner

Use this space for oversight and quality control, while keeping editing actions in Admin.
